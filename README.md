## Rodando o projeto

Clone the project and access the folder.

```bash
$ git clone git@github.com:vitorhugomarcal/ignews.git
$ cd ignews
```

Follow the steps below:

```bash
# Install the dependencies
$ yarn install
# Start the project
$ yarn dev
```

The app will be available for access on your browser
